# 📈 Stock Portfolio Tracker

This project is a simple **Stock Portfolio Tracker** developed using **Python** as part of my **CodeAlpha Internship**.

## 📌 Project Description

The program allows users to:
- Select stocks from a predefined list.
- Enter the quantity of shares.
- Calculate the investment value for each stock.
- Display the total portfolio investment value.

## 🚀 Features

- User-friendly console interface
- Hardcoded stock price dictionary
- Calculates total investment value
- Handles invalid stock names

## 🛠️ Technologies Used

- Python
- Dictionary
- Loops
- Conditional Statements
- Input/Output
- Basic Arithmetic

## ▶️ How to Run

1. Install Python.
2. Download or clone this repository.
3. Open the project folder in VS Code or Terminal.
4. Run the program:

```bash
python stock_portfolio_tracker.py
```
